# BusinessCard
I am creating a business card for myself, but with a unique design. I am making PCB that will host a webpage that will allow recruiters to easily see my information off of device. The device will also showcase an ECG reader to showcase my ability to route and work with sensors. These are all the files that pertain to this project and will contain KiCAD files along with SolidWorks of the chassis. 

Tech Build:
An AD832 analog front-end is read by an ESP32-WROOM-32E via its ADC. USB-C power siupplies an AMS1117 3.3V regulator; a CH340C USB-UART with interconnected DTR/RTS auto-reset manages programming. Electrodes are linked via a 3.5mm jack.
